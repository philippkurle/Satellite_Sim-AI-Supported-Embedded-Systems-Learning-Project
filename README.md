# Satellite_Sim - AI-Supported Embedded Systems Learning Project

This repository documents the Satellite_Sim project.

## Context

I started this project as a self-directed semester-break learning project after I was unable take an assistant position at a university lab, as I had originally planned.
The goal was to gain hands-on experience in embedded systems, hardware integration and system-oriented thinking, through the development of a simplified satellite subsystem simulator on an Arduino MEGA.
The software implementation was heavily AI-assisted, as I had very limited knowledge of C++ and object-oriented programming when I started.

## My Contribution

I focused on:

- defining the project concept and simplifying the satellite system idea
- selecting components and integrating the hardware
- using AI to generate and refine the software
- testing hardware behaviour and debugging the software with AI support
- documenting the project
- understanding and learning key parts of the software implementation

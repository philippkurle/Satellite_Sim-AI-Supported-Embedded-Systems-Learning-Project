# Satellite_Sim - AI-Supported Embedded Systems Learning Project

This repository contains the documentation of the Satellite_Sim project.

## Context

I started this project as a self-directed semester break learning project when I could not take an assistant position at a university lab, as I had originally planned.
The goal was to get first-hand experience in embedded systems, hardware integration and system-oriented thinking, through the development of a simplified satellite subsystem simulator on an Arduino MEGA.
The software implementation was heavily AI-assisted, since I had very limited C++/OOP knowledge when I started.

## My Contribution

I focused on:

- project concept, simplification of the satellite idea and component definition
- hardware setup and integration
- generating software with AI
- hardware behaviour testing and AI-assisted software debugging
- project documentation
- understanding and learning key parts of the software implementation
